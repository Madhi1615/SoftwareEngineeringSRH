Software Engineering — Day 1

From Yahoo → Google → Perplexity • Orkut → Instagram • Monoliths → AI Agents

🎓 Module 1: What Is Software Engineering?

Real-World Journey: Yahoo (human-curated) → Google (PageRank) → Perplexity (LLM + citations)

  • Define SE vs. programming
  • Ethics in AI: Bias, transparency, EU AI Act
  • Why SE matters: Therac-25, Boeing 737 MAX
🔥 2024 Trend: AI doesn’t replace engineers—it amplifies them. Your job: guide the AI.

🔄 Module 2: SDLC Models

Real-World Journey: Orkut (monolith) → Facebook (modular) → Instagram (microservices + events)

  • Waterfall → Agile → DevOps
  • Spotify’s Tribe/Guild model: Autonomy + alignment
  • AI in Agile: Forecast.app predicts sprint velocity
💡 Fun Fact: Spotify’s squads inspired Netflix, Uber, and Airbnb’s engineering culture.

📝 Module 3: Requirements in the AI Era

Real-World Journey: CRM forms → Salesforce Einstein (predictive AI)

  • Functional vs. non-functional (e.g., “cite sources” for Perplexity)
  • LLM prompt: “Generate user stories for a mental health chatbot”
  • Avoid vague: “The system shall be intelligent” → “LLM shall refuse medical advice”

🧱 Module 4: Design for Change

Real-World Model: Spotify’s Squads & Guilds

  • Cohesion (high) vs. Coupling (low)
  • KISS, DRY, YAGNI principles
  • Anti-pattern: “God Team” (one team owns everything)

🏗️ Module 5: Modern Architectures

Real-World Tool: n8n.io — Open-source workflow automation (like Zapier + AI)

  • Microservices vs. serverless
  • AI Agents as Services (e.g., “Research Agent”)
  • Patterns: Circuit Breaker → AI Fallback
🚀 Try it: Build a workflow: GitHub issue → Slack alert → Jira ticket (in 5 mins on n8n.cloud)

🧪 Module 6: AI-Augmented Testing

Real-World Tools: UiPath, Blue Prism (RPA for test automation)

  • LLMs generate edge-case test data
  • Non-deterministic AI outputs → semantic similarity checks
  • Self-healing tests with Applitools

🤝 Module 7: Git + AI Collaboration

Real-World Tools: GitHub Copilot, CodeRabbit

  • Copilot writes unit tests & PR descriptions
  • AI code reviewers flag security flaws
  • Risk: “AI debt” from over-reliance

🚀 Module 8: Capstone Kickoff

Build Your Own AI Agent!

  • Option 1: LLM Requirements Assistant (LangChain + OpenAI)
  • Option 2: RPA Bot for University Tasks (UiPath Community)
  • Option 3: Modernize a “Yahoo-era” monolith
🎯 You’ll learn: Prompt engineering, workflow automation, ethical AI, and teamwork!
