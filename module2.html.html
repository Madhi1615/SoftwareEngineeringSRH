Key Insight: Your SDLC model must match your scale, team size, and change velocity.

SDLC Models Overview (SWEBOK V4)

  • Waterfall: Linear, phase-gated. Works for stable requirements (e.g., defense systems). Fails with change.
  • Iterative: Build ‚Üí Evaluate ‚Üí Refine. NASA uses this for Mars rovers.
  • Spiral: Risk-driven. Used in high-stakes projects (e.g., medical devices).
  • Agile (Scrum/Kanban): Customer collaboration, working software, responding to change.
  • DevOps: Continuous integration, delivery, and feedback. GitHub deploys 100+ times/day.

Modern Twist: AI in SDLC

  • Forecast.app: Uses ML to predict sprint velocity and team capacity.
  • GitHub Copilot: Suggests code during construction, reducing cycle time.
  • AI Standups: Tools like Fireflies.ai summarize daily meetings and flag blockers.
